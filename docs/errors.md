## Базовый объект ошибки

Информация об ошибке при обработке запроса.

| Поле         | Тип       | Описание                                                                      | Обязательное | Пример                 |
|:-------------|:----------|:------------------------------------------------------------------------------|:-------------|:-----------------------|
| `statusCode` | `integer` | Числовой код ошибки, специфичный для приложения (не HTTP статус)              | Да           | 10101                  |
| `type`       | `string`  | Строковый идентификатор типа ошибки                                           | Да           | `FACE_SESSION_EXPIRED` |
| `data`       | `string`  | Дополнительные данные или контекст ошибки, может быть `null`                  | Да           | null                   |
| `message`    | `string`  | Человекочитаемое сообщение об ошибке на языке сессии (`ru`, `uz`, `oz`, `en`) | Да           | "Сообщение"            |

## 1. Базовые / Общие ошибки

Ошибки, которые могут возникнуть при работе с любым из сервисов (AbleID, P2P, Pay).

| StatusCode | Type                                   | Описание                                          |
|:-----------|:---------------------------------------|:--------------------------------------------------|
| **10100**  | `FACE_LIMIT_HAS_BEEN_REACHED`          | Превышен лимит запросов                           |
| **10101**  | `FACE_SESSION_EXPIRED`                 | Сессия истекла (повторная попытка слишком поздно) |
| **10102**  | `FACE_INVALID_URL`                     | Неверный URL (например, в хуках)                  |
| **10301**  | `FACE_TRANSACTION_FAILED_VERIFICATION` | Транзакция существует, но не прошла верификацию   |
| **20000**  | `SERVICE_TEMPORARILY_UNAVAILABLE`      | Сервис временно не доступен, попробуйте позже     |
| **20002**  | `ERROR_REQUEST_PROCESSING`             | Ошибка в обработке запроса                        |
| **20004**  | `EMPTY_DATA`                           | Данные не найдены во внешнем сервисе              |
| **20202**  | `INCORRECT_DATA_FORMAT`                | Некорректные данные                               |

## 2. AbleID (Идентификация)

### [POST] /check/start/second (Резиденты)

Ошибки, возникающие при старте процесса идентификации резидента.

| StatusCode | Type                          | Описание                           |
|:-----------|:------------------------------|:-----------------------------------|
| **10103**  | `FACE_INCORRECT_PINFL`        | Некорректный ПИНФЛ                 |
| **10105**  | `FACE_INCORRECT_PINFL_DATE`   | Некорректная дата рождения и ПИНФЛ |
| **20005**  | `EMPTY_DATA_ADDRESS`          | Данные прописки не найдены         |
| **20201**  | `NO_MANDATORY_DATA_AVAILABLE` | Обязательные данные отсутствуют    |

### [POST] /check/start/foreign (Нерезиденты)

Ошибки, возникающие при старте процесса идентификации нерезидента.

| StatusCode | Type                  | Описание                                                   |
|:-----------|:----------------------|:-----------------------------------------------------------|
| **10200**  | `FACE_DATA_NOT_FOUND` | Пользователь не найден (или данные документа не совпадают) |

## 5. Получение данных (Webhooks)

Ошибки, возникающие при запросе данных через webhook-эндпоинты. Данные отправляются на заранее прописанные статические URL.

> **Напоминание:** Все эндпоинты ниже могут вернуть общие ошибки из этого раздела, а также ошибки из раздела [Базовые / Общие ошибки](#1-базовые--общие-ошибки). Специфичные ошибки указаны отдельно для каждого эндпоинта.

### Общие ошибки всех webhook-эндпоинтов

| StatusCode | Type                         | Описание                                                                                |
|:-----------|:-----------------------------|:----------------------------------------------------------------------------------------|
| **10101**  | `FACE_SESSION_EXPIRED`       | Сессия истекла (превышено время жизни сессии)                                           |
| **10202**  | `FACE_PROJECT_NOT_FOUND`     | Проект не найден (неверный `projectId` или `secret`)                                    |
| **10302**  | `FACE_TRANSACTION_NOT_FOUND` | Сессия не найдена (неверный `attemptId` или не принадлежит проекту)                     |
| **10500**  | `FACE_INTERNAL_ERROR`        | Внутренняя ошибка сервера                                                               |
| **20423**  | `ONEID_DATA_LOCKED`          | Передача персональных данных заблокирована в системе OneID                              |

### [POST] /check/get_passport

Получение паспортных данных резидента.

| StatusCode | Type                     | Описание                                   |
|:-----------|:-------------------------|:-------------------------------------------|
| **10103**  | `FACE_INCORRECT_PINFL`   | Некорректный ПИНФЛ (не прошёл валидацию)   |
| **10204**  | `FACE_PASSPORT_EXPIRED`  | Паспорт истёк (дата окончания в прошлом)   |

### [POST] /check/get_person

Получение полной информации о резиденте (паспорт + прописка).

| StatusCode | Type                       | Описание                                        |
|:-----------|:---------------------------|:------------------------------------------------|
| **10103**  | `FACE_INCORRECT_PINFL`     | Некорректный ПИНФЛ (не прошёл валидацию)        |
| **10104**  | `FACE_INCORRECT_DOCUMENT`  | Данные человека не найдены по указанному ПИНФЛ  |
| **10204**  | `FACE_PASSPORT_EXPIRED`    | Паспорт истёк (дата окончания в прошлом)        |

### [POST] /check/get_foreign_person

Получение данных пересечения нерезидента.

Специфичных ошибок нет — применяются только [общие ошибки webhook-эндпоинтов](#общие-ошибки-всех-webhook-эндпоинтов).

### [POST] /check/get_photo

Получение фотографии пользователя.

| StatusCode | Type                                   | Описание                                         |
|:-----------|:---------------------------------------|:-------------------------------------------------|
| **10203**  | `FACE_NOT_VALID`                       | Фотография не найдена или недоступна             |

### [POST] /check/registration

Получение данных прописки резидента.

| StatusCode | Type                          | Описание                                 |
|:-----------|:------------------------------|:-----------------------------------------|
| **10103**  | `FACE_INCORRECT_PINFL`        | Некорректный ПИНФЛ (не прошёл валидацию) |
| **10205**  | `FACE_REGISTRATION_NOT_FOUND` | Прописка отсутствует (данные не найдены) |
| **20005**  | `EMPTY_DATA_ADDRESS`          | Данные прописки не найдены               |

### [POST] /check/get_work

Получение данных о текущем месте работы.

| StatusCode | Type                          | Описание                                           |
|:-----------|:------------------------------|:---------------------------------------------------|
| **10103**  | `FACE_INCORRECT_PINFL`        | Некорректный ПИНФЛ (не прошёл валидацию)           |

## Примеры ответов с ошибками

### Пример 1: Сессия истекла

**HTTP Status:** 400

```json
{
  "data": null,
  "statusCode": 10101,
  "type": "FACE_SESSION_EXPIRED",
  "message": "Сессия истекла"
}
```

### Пример 2: Передача данных заблокирована в OneID

**HTTP Status:** 423

```json
{
  "data": null,
  "statusCode": 20423,
  "type": "ONEID_DATA_LOCKED",
  "message": "Передача персональных данных заблокирована. Для использования сервиса AbleID необходимо активировать передачу персональных данных в системе OneID"
}
```

### Пример 3: Внутренняя ошибка

**HTTP Status:** 500

```json
{
  "data": null,
  "statusCode": 10500,
  "type": "FACE_INTERNAL_ERROR",
  "message": "Внутренняя ошибка сервера. Повторите попытку позже."
}
```
