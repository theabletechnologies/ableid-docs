## Базовый объект ошибки

Информация об ошибке при обработке запроса.

| Поле         | Тип       | Описание                                                         | Обязательное | Пример                 |
| :----------- | :-------- | :--------------------------------------------------------------- | :----------- | :--------------------- |
| `statusCode` | `integer` | Числовой код ошибки, специфичный для приложения (не HTTP статус) | Да           | 10101                  |
| `type`       | `string`  | Строковый идентификатор типа ошибки                              | Да           | `FACE_SESSION_EXPIRED` |
| `data`       | `string`  | Дополнительные данные или контекст ошибки, может быть `null`     | Да           | null                   |
| `message`    | `string`  | Человекочитаемое сообщение об ошибке (на русском языке)          | Да           | "Сообщение"            |

---

## 1. Базовые / Общие ошибки

Ошибки, которые могут возникнуть при работе с любым из сервисов (AbleID, P2P, Pay).

| StatusCode | Type                                   | Описание                                          |
| :--------- | :------------------------------------- | :------------------------------------------------ |
| **10100**  | `FACE_LIMIT_HAS_BEEN_REACHED`          | Превышен лимит запросов                           |
| **10101**  | `FACE_SESSION_EXPIRED`                 | Сессия истекла (повторная попытка слишком поздно) |
| **10102**  | `FACE_INVALID_URL`                     | Неверный URL (например, в хуках)                  |
| **10301**  | `FACE_TRANSACTION_FAILED_VERIFICATION` | Транзакция существует, но не прошла верификацию   |
| **20000**  | `SERVICE_TEMPORARILY_UNAVAILABLE`      | Сервис временно не доступен, попробуйте позже     |
| **20002**  | `ERROR_REQUEST_PROCESSING`             | Ошибка в обработке запроса                        |
| **20004**  | `EMPTY_DATA`                           | Данные не найдены во внешнем сервисе              |
| **20202**  | `INCORRECT_DATA_FORMAT`                | Некорректные данные                               |

---

## 2. AbleID (Идентификация)

### [POST] /check/start/second (Резиденты)

Ошибки, возникающие при старте процесса идентификации резидента.

| StatusCode | Type                          | Описание                           |
| :--------- | :---------------------------- | :--------------------------------- |
| **10103**  | `FACE_INCORRECT_PINFL`        | Некорректный ПИНФЛ                 |
| **10105**  | `FACE_INCORRECT_PINFL_DATE`   | Некорректная дата рождения и ПИНФЛ |
| **20005**  | `EMPTY_DATA_ADDRESS`          | Данные прописки не найдены         |
| **20201**  | `NO_MANDATORY_DATA_AVAILABLE` | Обязательные данные отсутствуют    |

### [POST] /check/start/foreign (Нерезиденты)

Ошибки, возникающие при старте процесса идентификации нерезидента.

| StatusCode | Type                  | Описание                                                   |
| :--------- | :-------------------- | :--------------------------------------------------------- |
| **10200**  | `FACE_DATA_NOT_FOUND` | Пользователь не найден (или данные документа не совпадают) |

## 3. Able P2P (Верификация)

Сервисы верификации используют схожую логику с идентификацией, но с контекстом проверки.

### [POST] /check/start/verification/second (Резиденты)

| StatusCode | Type                          | Описание                           |
| :--------- | :---------------------------- | :--------------------------------- |
| **10103**  | `FACE_INCORRECT_PINFL`        | Некорректный ПИНФЛ                 |
| **10105**  | `FACE_INCORRECT_PINFL_DATE`   | Некорректная дата рождения и ПИНФЛ |
| **20005**  | `EMPTY_DATA_ADDRESS`          | Данные прописки не найдены         |
| **20201**  | `NO_MANDATORY_DATA_AVAILABLE` | Обязательные данные отсутствуют    |

### [POST] /check/start/verification/foreign (Нерезиденты)

| StatusCode | Type                  | Описание                                     |
| :--------- | :-------------------- | :------------------------------------------- |
| **10200**  | `FACE_DATA_NOT_FOUND` | Пользователь не найден (несовпадение данных) |
