Ошибки

Базовый объект ошибки​

Информация об ошибке при обработке запроса.

ПолеТипОписаниеОбязательноеПример
statusCodeintegerЧисловой код ошибки, специфичный для приложения (не HTTP статус)Да10101
typestringСтроковый идентификатор типа ошибкиДаFACE_SESSION_EXPIRED
datastringДополнительные данные или контекст ошибки, может быть nullДаnull
messagestringЧеловекочитаемое сообщение об ошибке на языке сессии (ru, uz, oz, en)Да"Сообщение"

1. Базовые / Общие ошибки​

Ошибки, которые могут возникнуть при работе с любым из сервисов (AbleID, P2P, Pay).

StatusCodeTypeОписание
10100FACE_LIMIT_HAS_BEEN_REACHEDПревышен лимит запросов
10101FACE_SESSION_EXPIREDСессия истекла (повторная попытка слишком поздно)
10102FACE_INVALID_URLНеверный URL (например, в хуках)
10301FACE_TRANSACTION_FAILED_VERIFICATIONТранзакция существует, но не прошла верификацию
20000SERVICE_TEMPORARILY_UNAVAILABLEСервис временно не доступен, попробуйте позже
20002ERROR_REQUEST_PROCESSINGОшибка в обработке запроса
20004EMPTY_DATAДанные не найдены во внешнем сервисе
20202INCORRECT_DATA_FORMATНекорректные данные

2. AbleID (Идентификация)​

[POST] /check/start/second (Резиденты)​

Ошибки, возникающие при старте процесса идентификации резидента.

StatusCodeTypeОписание
10103FACE_INCORRECT_PINFLНекорректный ПИНФЛ
10105FACE_INCORRECT_PINFL_DATEНекорректная дата рождения и ПИНФЛ
20005EMPTY_DATA_ADDRESSДанные прописки не найдены
20201NO_MANDATORY_DATA_AVAILABLEОбязательные данные отсутствуют

[POST] /check/start/foreign (Нерезиденты)​

Ошибки, возникающие при старте процесса идентификации нерезидента.

StatusCodeTypeОписание
10200FACE_DATA_NOT_FOUNDПользователь не найден (или данные документа не совпадают)

5. Получение данных (Webhooks)​

Ошибки, возникающие при запросе данных через webhook-эндпоинты. Данные отправляются на заранее прописанные статические URL.

Напоминание: Все эндпоинты ниже могут вернуть общие ошибки из этого раздела, а также ошибки из раздела Базовые / Общие ошибки. Специфичные ошибки указаны отдельно для каждого эндпоинта.

Общие ошибки всех webhook-эндпоинтов​

StatusCodeTypeОписание
10101FACE_SESSION_EXPIREDСессия истекла (превышено время жизни сессии)
10202FACE_PROJECT_NOT_FOUNDПроект не найден (неверный projectId или secret)
10302FACE_TRANSACTION_NOT_FOUNDСессия не найдена (неверный attemptId или не принадлежит проекту)
10500FACE_INTERNAL_ERRORВнутренняя ошибка сервера
20423ONEID_DATA_LOCKEDПередача персональных данных заблокирована в системе OneID

[POST] /check/get_passport​

Получение паспортных данных резидента.

StatusCodeTypeОписание
10103FACE_INCORRECT_PINFLНекорректный ПИНФЛ (не прошёл валидацию)
10204FACE_PASSPORT_EXPIREDПаспорт истёк (дата окончания в прошлом)

[POST] /check/get_person​

Получение полной информации о резиденте (паспорт + прописка).

StatusCodeTypeОписание
10103FACE_INCORRECT_PINFLНекорректный ПИНФЛ (не прошёл валидацию)
10104FACE_INCORRECT_DOCUMENTДанные человека не найдены по указанному ПИНФЛ
10204FACE_PASSPORT_EXPIREDПаспорт истёк (дата окончания в прошлом)

[POST] /check/get_foreign_person​

Получение данных пересечения нерезидента.

Специфичных ошибок нет — применяются только общие ошибки webhook-эндпоинтов.

[POST] /check/get_photo​

Получение фотографии пользователя.

StatusCodeTypeОписание
10203FACE_NOT_VALIDФотография не найдена или недоступна

[POST] /check/registration​

Получение данных прописки резидента.

StatusCodeTypeОписание
10103FACE_INCORRECT_PINFLНекорректный ПИНФЛ (не прошёл валидацию)
10205FACE_REGISTRATION_NOT_FOUNDПрописка отсутствует (данные не найдены)
20005EMPTY_DATA_ADDRESSДанные прописки не найдены

[POST] /check/get_work​

Получение данных о текущем месте работы.

StatusCodeTypeОписание
10103FACE_INCORRECT_PINFLНекорректный ПИНФЛ (не прошёл валидацию)

Примеры ответов с ошибками​

Пример 1: Сессия истекла​

HTTP Status: 400

{
  "data": null,
  "statusCode": 10101,
  "type": "FACE_SESSION_EXPIRED",
  "message": "Сессия истекла"
}

Пример 2: Передача данных заблокирована в OneID​

HTTP Status: 423

{
  "data": null,
  "statusCode": 20423,
  "type": "ONEID_DATA_LOCKED",
  "message": "Передача персональных данных заблокирована. Для использования сервиса AbleID необходимо активировать передачу персональных данных в системе OneID"
}

Пример 3: Внутренняя ошибка​

HTTP Status: 500

{
  "data": null,
  "statusCode": 10500,
  "type": "FACE_INTERNAL_ERROR",
  "message": "Внутренняя ошибка сервера. Повторите попытку позже."
}
